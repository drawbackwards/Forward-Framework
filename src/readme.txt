== Description ==

A modern WordPress starter theme built for designers. Many themes try to do everything, Forward doesn't. Meet a lightweight, WordPress starter theme that gives you a quality jump-start on your next project.

== Setup ==

1. Install and activate Forward.
2. Create a new menu and assign to the 'Primary Menu' theme location.

== Frequently Asked Questions ==

= How do I change custom web fonts? =

Custom web fonts are enqueued using https://www.google.com/fonts

1. Open functions.php
2. Search for forward_google_fonts()
3. Modify $query_args
